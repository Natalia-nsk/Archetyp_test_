# Archetyp_test_
Archetyp_test_ is an educational project where the user chooses his archetype by answering 8 questions with 3 options, animation is used separately. The goal is to demonstrate the architecture and logic of the process.

Концепт 
Тест "Какой вы архетип по системе Карла Юнга?" 

На основе вопросов -> происходит подсчет ->  баллов, в результате которых выводится на экран -> результат, основанный на -> описании архетипов.

Данные не сохраняются - нет аккаунтов, истории, сравнения, cookies, баз данных.

Используется JSON для:
- вопросы для теста; 
- логика подсчета баллов; 
- описание архетипов; 
- визуализация;

Визуализация представлена:
статичный фон прохождения теста; 
анимации при завершении теста -> пользователь узнает свой главный архетип, затем выводятся три дополняющих. 

Инструменты: 
Python
Streamlit
JSON
Модульная архитектура.

Проект пока не выложен и существует как концепт.

AI использовался в работе как инструмент.
